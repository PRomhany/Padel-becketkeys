
Becket Keys Padel Web App

This is the corrected web project for GitHub and Vercel.

How to use:
1. Unzip this folder.
2. Upload the contents to a GitHub repository.
3. Import that repository into Vercel.
4. Click Deploy.

Important:
- This version is a standalone trial build.
- Bookings are stored in the browser local storage, so they are not shared between different staff devices yet.
- If you want shared live bookings for all staff, the next step is a Supabase-backed version.
